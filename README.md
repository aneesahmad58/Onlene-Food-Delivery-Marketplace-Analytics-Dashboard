# Onlene Food Delivery Marketplace Analytics Dashboard

## Project Overview
This project simulates a real-world food delivery platform (similar to Zomato/Swiggy) and analyzes marketplace performance using SQL and Power BI.

## Key Features
- GMV, Net GMV, Platform Revenue, AOV
- City-wise performance analysis
- Top Restaurants ranking
- Discount vs No Discount impact
- Customer Ratings distribution
- Interactive Power BI Dashboard

## Tools Used
- SQL Server
- Power BI
- DAX
- Data Modeling

## Files Included
- Power BI Dashboard (.pbix)
- Dataset files (.csv)
- Dashboard screenshots

## Business Use Case
Helps in analyzing revenue performance, growth trends, operational metrics, and customer experience insights for decision-making.
